# Data Extraction Protocol

Add one row per sample/process condition.

Required fields: source_id, DOI, sample_id, alloy_system, laser_power_W, scan_speed_mm_s, hatch_spacing_mm, layer_thickness_mm, VED_J_mm3, powder_Ni_at_pct, measured_Ni_at_pct, heat treatment, density, porosity, Ms, Mf, As, Af, hysteresis, recoverable strain, residual strain, UTS, elongation, phases, precipitates and data_quality.

Use data_quality labels: exact_table, exact_text, digitized_figure, author_request, inferred_rule, demo_synthetic.

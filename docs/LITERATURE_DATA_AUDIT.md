# Literature Data Audit

The repository separates exact literature facts from demo rows.

- `data/exact_literature_process_rows.csv`: exact A1-D4 process rows and sample observations from an open LPBF Ni-rich NiTi study.
- `data/paper_level_numeric_facts.csv`: exact paper-level constants, ranges, ML benchmark facts and performance targets.
- `data/demo_training_seed.csv`: synthetic demo data for testing the ML interface only.

Do not report metrics from demo rows as scientific validation.
